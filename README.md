# Bank Management System (C++)

## Features
- Admin Login
- User Login
- Create Account
- Deposit Money
- Withdraw Money
- Transfer Funds
- Search Customer
- File Handling using Binary Files

## Technologies Used
- C++
- File Handling
- Object-Oriented Programming

## How to Run
Compile the project:
g++ Banking_Application.cpp -o Banking_Application

Run:
Banking_Application.exe
